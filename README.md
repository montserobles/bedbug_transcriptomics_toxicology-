# Bed Bug Transcriptomics Toxicology

RNA-seq analysis pipeline and visualization scripts associated with the study:

“Transcriptomic responses of pyrethroid-susceptible and -resistant *Cimex lectularius* strains to deltamethrin, piperonyl butoxide, and essential oil constituents.”

## Overview

This repository contains scripts and metadata used for differential gene expression analyses of susceptible (Harlan) and resistant (Knoxville) bed bug strains following sublethal exposure to synthetic insecticides and essential oil constituents.

Analyses include:
- DESeq2 differential expression analysis
- PCA visualization
- Volcano plots
- UpSet overlap analyses
- Functional annotation workflows

## Treatments
- Deltamethrin
- Piperonyl butoxide (PBO)
- Thymol
- Carvacrol

## Software
- R
- DESeq2
- ggplot2
- UpSetR

## Data Availability
Raw RNA-seq data are available through NCBI SRA under BioProject:  PRJNA1460368
